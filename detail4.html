Traveling Website Project

Technologies Used:
  • Laravel 9 (Backend)
  • HTML5, CSS3, JavaScript (Frontend)
  • Bootstrap 5
  • Figma (UI Design)
Project Details:

This is my second major project where I developed a traveling website for my older brother. The goal of the website is to document and share travel experiences using his personal car in an elegant and informative interface.

I designed the UI in Figma, then built the frontend using HTML5, CSS3, JavaScript, and Bootstrap 5. The backend is built with Laravel 9 to manage destinations, dynamic content, and user interaction.
