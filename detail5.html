Warkop Hamzah Inventory System

Technologies Used:
  • Laravel 9
  • HTML5 & CSS3
  • JavaScript
  • Bootstrap 5
  • Figma (UI/UX Design)
Project Details:

This project was built for a small coffee shop called Warkop Hamzah. I developed a custom inventory and transaction management system to help the owner monitor product stock, manage customer orders, and track sales.

The frontend was designed using Figma and implemented using HTML, CSS, Bootstrap, and JavaScript. The backend was developed with Laravel 9 to handle CRUD operations, user authentication, transaction logging, and stock reports.
